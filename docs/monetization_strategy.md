# Стратегия монетизации Elemental Fae

## Модель: Freemium + Battle Pass + Реклама

## 1. Внутриигровые покупки (IAP)
- **Кристаллы** (премиум-валюта): $0.99–$99.99
- **Приманки стихий:** повышают шанс встретить редкую фею
- **Косметика:** скины для фей, эффекты, аватары
- **Инвентарь:** расширение слотов для фей и предметов

## 2. Battle Pass (сезонный пропуск)
- **Цена:** $9.99 / месяц
- **Награды:** эксклюзивная фея, повышенный шанс редких эссенций, дополнительные задания
- **Прогноз конверсии:** 5–7% активных игроков

## 3. Реклама
- **Rewarded Video:** просмотр рекламы за кристаллы (~$0.02 за просмотр)
- **Баннеры:** только в меню (не в геймплее)
- **Спонсорские зоны:** брендированные геозоны

## 4. Партнёрства
- **Фитнес-бренды:** скидки за шаги (Nike, Adidas)
- **Городские власти:** туристические маршруты с феями
- **Кафе и рестораны:** эксклюзивные феи в заведениях

## 5. Прогноз доходов (на 100,000 DAU)
- IAP: ~$450,000 / месяц
- Battle Pass: ~$50,000 / месяц
- Реклама: ~$30,000 / месяц
- **Итого:** ~$530,000 / месяц
