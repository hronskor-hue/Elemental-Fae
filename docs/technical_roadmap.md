# Техническая дорожная карта Elemental Fae

## Фаза 1: Прототип (месяцы 1–3)
- [ ] Готовый лендинг (index.html)
- [ ] Unity-проект с базовой картой и персонажем
- [ ] Интеграция Google Fit / Apple HealthKit
- [ ] Прототип спавна существ в геозонах
- [ ] Серверный прототип (REST API + PostgreSQL)

## Фаза 2: MVP (месяцы 4–8)
- [ ] 20+ игровых зон в 1 городе
- [ ] 7 типов фей (по 3 стадии эволюции)
- [ ] Система крафта (приманки, кристаллы)
- [ ] Механика «недель стихий»
- [ ] Система аккаунтов и прогресса
- [ ] Античит-система (геозоны + шаги)

## Фаза 3: Бета-тестирование (месяцы 9–12)
- [ ] Закрытое бета-тестирование (1000+ пользователей)
- [ ] Балансировка игровой экономики
- [ ] Система репортов и модерации
- [ ] Интеграция рекламы и магазина
- [ ] Подготовка к запуску в App Store / Google Play

## Фаза 4: Запуск (месяцы 12–15)
- [ ] Глобальный запуск в App Store и Google Play

- [ ] PvP-арены

- [ ] Партнёрства с брендами

## Фаза 5: Развитие (месяцы 15+)
- [ ] Гильдии и кланы
- [ ] Рейды на легендарных фей
- [ ] NFT-интеграция (опционально)
