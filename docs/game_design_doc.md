# Game Design Document: Elemental Fae

## 1. Концепция игры
Мобильная геолокационная RPG с коллекционированием магических существ (фей). Игроки исследуют реальный мир, находят и приручают фей семи стихий, создают альянсы и участвуют в сезонных событиях.

## 2. Стихии и существа
| Стихия | Цвет | Среда обитания | Легендарная фея |
|--------|------|----------------|-----------------|
| 🔥 Огонь | Красный | Город, площади | Феникс |
| 💧 Вода | Синий | Реки, озёра | Левиафан |
| 🌿 Природа | Зелёный | Парки, леса | Энт |
| 🌪️ Воздух | Жёлтый | Открытые пространства | Грифон |
| 🪨 Камень | Коричневый | Горы, скалы | Голем |
| ✨ Свет | Белый | Памятники, церкви | Серафим |
| 🌑 Тьма | Фиолетовый | Ночные локации | Теневой Дракон |

## 3. Механики
- **Поиск:** Существа появляются в геозонах соответственно стихии.
- **Приручение:** Мини-игра с использованием кристаллов (аналог покеболов).
- **Прокачка:** Опыт за шаги, бои и кормление.
- **Эволюция:** 3 стадии для каждой феи.
- **Крафт:** Артефакты из ресурсов (эссенций, пыльцы, минералов).
- **Недели стихий:** Каждую неделю бонус к спавну одной стихии.
- **PvP-арены:** Сражения 3v3 в определённых геозонах.

## 4. Игровая экономика
| Ресурс | Способ получения | Использование |
|--------|------------------|---------------|
| Эссенции | Прогулки в зонах стихий | Крафт, эволюция |
| Кристаллы | Ежедневные задания | Приручение, PvP |
| Шаги | Ходьба (Google Fit / HealthKit) | Открытие сундуков |
| Монеты | Покупка / задания | Магазин |
