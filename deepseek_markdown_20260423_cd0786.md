# Риски и трудности Elemental Fae

## 1. Технические риски

| Риск | Вероятность | Влияние | Меры предотвращения |
|------|-------------|---------|---------------------|
| GPS-спуфинг | Высокая | Критичное | Двойная система: геозоны + шагомер. Серверная валидация движения на аномалии. |
| Высокое энергопотребление | Высокая | Среднее | Оптимизация GPS-запросов, Fused Location Provider, режим энергосбережения. |
| Ошибки определения локации | Средняя | Среднее | Буферизация координат, минимальное время внутри зоны для засчитывания. |
| Масштабирование серверов | Средняя | Высокое | Облачная архитектура с автоскейлингом, CDN для статики. |

## 2. Юридические риски

| Риск | Вероятность | Влияние | Меры предотвращения |
|------|-------------|---------|---------------------|
| Приватность (геоданные) | Средняя | Высокое | Соответствие GDPR/CCPA. Анонимизация данных, политика конфиденциальности. |
| Ответственность за травмы | Низкая | Критичное | Пользовательское соглашение, предупреждения в интерфейсе, только безопасные зоны. |
| Авторские права (дизайн фей) | Низкая | Среднее | Оригинальный арт, юридическая проверка концептов. |
| Регулирование в разных странах | Средняя | Среднее | Локализация правил игры под законы страны. |

## 3. Бизнес-риски

| Риск | Вероятность | Влияние | Меры предотвращения |
|------|-------------|---------|---------------------|
| Конкуренция с Pokémon GO | Высокая | Высокое | Уникальные механики (недели стихий, крафт). Нишевая аудитория. |
| Низкая вовлечённость | Средняя | Высокое | Сильная система онбординга, ежедневные задания, социальные механики. |
| Проблемы монетизации | Средняя | Среднее | Гибридная модель (IAP + реклама + партнёрства). |

## 4. Командные риски
- Отсутствие ключевых специалистов (Unity, бэкенд).
- **Решение:** Начать с поиска core-команды, рассмотреть аутсорсинг.
