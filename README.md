## О проекте

Elemental Fae — мобильная геолокационная игра с элементами AR, объединяющая механику шагомера и коллекционирования магических существ. Игроки исследуют реальный мир, открывают и собирают фей семи стихий, создают альянсы и соревнуются за контроль над магическими локациями.

### Ключевые механики
- 🧚 **Коллекционирование фей** 7 стихий (Огонь, Вода, Природа, Воздух, Камень, Свет, Тьма)
- 👣 **Двойной контроль активности** — шаги + геопозиция (античит)
- 🗺️ **Ручное выделение игровых зон** разработчиком (без ИИ, высокая безопасность)
- ⚗️ **Крафт** артефактов и приманок из найденных ресурсов
- 📅 **Недели стихий** (механика «то густо, то пусто» из Heroes III)

### Технологии
- **Клиент:** Unity / Unreal Engine (мобильная версия) + ARCore/ARKit
- **Сервер:** Go / Node.js, PostgreSQL, Redis
- **Карты:** Mapbox / Google Maps Platform
- **Шагомер:** Google Fit API / Apple HealthKit
- **Хостинг:** GitHub Pages (лендинг), AWS/Google Cloud (игровой сервер)
