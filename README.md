## 🔮 Ключевые особенности

| 🧚 Коллекционирование | 👣 Двойной контроль | 🗺️ Игровые зоны |
|----------------------|---------------------|------------------|
| 7 стихий, эволюция до легендарных форм. Каждая фея уникальна и привязана к реальной местности. | **Шагомер + геолокация:** чтобы получить редкую фею, нужно и находиться в правильной зоне, и пройти внутри неё заданное количество шагов. Античит на системном уровне. | Разработчик вручную выделяет безопасные полигоны (парки, скверы, набережные). Никакого ИИ — только ручная модерация. |

| ⚗️ Крафт | 📅 То густо, то пусто | 💰 Бизнес-модель |
|-----------|-----------------------|-------------------|
| Собирай эссенции, пыльцу и осколки шагов. Создавай приманки, кристаллы эволюции и переносные алтари. | **Как в Heroes III!** Каждую неделю одна стихия получает бонус к спавну по всей карте. Следи за календарём, чтобы не пропустить свою. | Freemium + Battle Pass. Подписка через ЮKassa (54-ФЗ), внутриигровая валюта, партнёрства с городскими парками. |

## 🛠️ Технологический стек

<div align="center">

| Компонент | Решение |
|-----------|---------|

| 🗺️ Карты | Яндекс.Карты API / MapKit |
| ☁️ Бэкенд | Yandex Cloud Functions (Python) + YDB |
